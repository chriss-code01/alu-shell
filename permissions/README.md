# Shell Permissions Project

This project contains Bash scripts demonstrating Linux user management, file ownership, group ownership, and permission management.

## Scripts

* 0-iam_betty: Switch current user to betty
* 1-who_am_i: Display current username
* 2-groups: Display current user's groups
* 3-new_owner: Change file owner to betty
* 4-empty: Create an empty file
* 5-execute: Add execute permission to owner
* 6-multiple_permissions: Add execute permissions and read permissions
* 7-everybody: Add execute permission for everyone
* 8-James_Bond: Set permissions to 007
* 9-John_Doe: Set permissions to 753
* 10-mirror_permissions: Copy permissions from another file
* 11-directories_permissions: Add execute permission to directories only
* 12-directory_permissions: Create directory with 751 permissions
* 13-change_group: Change group owner
* 14-change_owner_and_group: Change owner and group for all files
* 15-symbolic_link_permissions: Change symbolic link ownership
* 16-if_only: Change ownership only if owned by guillaume
